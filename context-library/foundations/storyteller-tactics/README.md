# Storyteller Tactics

Personal reference corpus from *Storyteller Tactics* (Steve Rawling / [Pip Decks](https://pipdecks.com/storytellertactics)). Use when building, shaping, or critiquing narratives: pitches, design critiques, stakeholder presentations, research share-outs, launch stories, and idea selling.

**Not for redistribution.** Cards retain Pip Decks attribution and © lines. Design OS owns this index and the `/storytelling` skill wrapper only.

The copilot and `/storytelling` skill pull from here for narrative craft questions.

---

## How to Use

1. Open this README.
2. If the goal is clear (convince, sell, lead, etc.), jump to a [Recipe](#recipe).
3. Otherwise walk the [Story Building System](#story-building-system).
4. For a fast path, use the [Desert Island seven](#desert-island-cards).

Full flowchart and category intros: [`story-building-system.md`](story-building-system.md). How to approach the deck: [`instruction-card.md`](instruction-card.md). Credits: [`credits.md`](credits.md).

## Story Building System

Decision path through the deck:

1. **Why do you need a story?** No → [Concept](#concept)
2. **Where will you find it?** No → [Explore](#explore)
3. **What's your role?** No → [Character](#character)
4. **What must the story do?** No → [Function](#function)
5. **How will you plan it?** No → [Structure](#structure)
6. **How will you tell it?** No → [Style](#style)
7. **How will you share it?** No → [Organise](#organise)

Or combine tactics with a [Recipe](#recipe).
