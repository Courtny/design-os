# Outputs

All active work created by Claude goes here first. Once a document is finalized, move it to the appropriate `context-library/` folder for reference.

This keeps active drafts separate from your canonical knowledge base.

---

## Subfolders

| Folder | What goes here |
|--------|---------------|
| `briefs/` | UX briefs in progress |
| `design-specs/` | Specs being drafted or in review |
| `prototypes/` | HTML/CSS/JS prototypes from `/prototype` |
| `critique-notes/` | Notes from design critiques |
| `research-synthesis/` | Processed research and session summaries |
| `meeting-notes/` | Meeting notes from current syncs |
| `decisions/` | Decision docs being drafted |
| `mcp-integration-logs/` | Logs from MCP connections |
